# Livraria

Projeto de uma **livraria online** com funcionalidades de tema claro/escuro, favoritos e personalização da capa dos livros.

## Funcionalidades

1. **Tema Claro e Escuro**

   * Permite alternar entre o modo claro e escuro para melhor experiência do usuário.

2. **Adicionar aos Favoritos**

   * Usuários podem salvar livros favoritos para acesso rápido.

3. **Adicionar Capa do Livro**

   * Permite ao usuário adicionar ou atualizar a capa de cada livro na livraria.

## Como usar

### Frontend

1. Navegue até a pasta do frontend:

   ```bash
   cd frontend
   ```
2. Instale as dependências:

   ```bash
   npm install
   ```
3. Execute o frontend em modo de desenvolvimento:

   ```bash
   npm run dev
   ```

### Backend

1. Navegue até a pasta do backend:

   ```bash
   cd backend
   ```
2. Instale as dependências:

   ```bash
   npm install
   ```
3. Execute o backend em modo de desenvolvimento:

   ```bash
   npm run dev
   ```

## Observações

* Certifique-se de que o backend esteja rodando antes de iniciar o frontend.
* Para personalizar capas e favoritos, você precisa estar logado (caso o sistema tenha autenticação).
